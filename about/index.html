About Me#

「所有複雜系統的底層，最終都是狀態機與反饋環的組合。」

我是一名在 複雜系統 (Complex Systems) 與 智能體 (Intelligent Agents) 之間尋找規律的工程師。

我的核心驅動力來自於一種強烈的對比：一方面是 Linux Kernel 級別的絕對確定性 (Determinism)，另一方面是 LLM 湧現出的概率性智能 (Probabilistic Intelligence)。我的目標是構建一座橋樑，利用 SRE 的工程嚴謹度，將 AI 從「不可預測的聊天機器人」轉化為「可可靠、可觀測、可演進的生產力系統」。

🚀 演進軌跡 (The Trajectory)#

我將自己的成長視為一次持續的「能力合成」過程：

🔹 Phase 1: Infrastructure Depth (穩定性的基石)#

深耕於 DevOps 與 SRE 領域。我對系統的理解始於對 $\text{Packet Loss}$ 的追蹤、對 $\text{eBPF}$ 的挖掘以及對 $\text{K8s}$ 網路底層的解構。這一階段我學會了如何與最底層的硬體和內核對話，建立起對穩定性 (Stability) 和可觀測性 (Observability) 的極致追求。

🔹 Phase 2: Intelligence Breadth (認知維度的擴張)#

當我發現傳統的確定性自動化無法解決複雜的認知任務時，我將視角轉向 AI Engineering。我開始探索如何將 LLM 作為核心推理引擎，並圍繞它構建 $\text{RAG}$ 管道與 $\text{Agentic Workflows}$。這是我從「管理資源」轉向「管理智能」的轉折點。

🔹 Phase 3: Digital Twin Synthesis (數位雙生的合成)#

目前，我正致力於構建我的「數位雙生」知識體系。這不單是筆記的累積，而是一次知識的工程化：將碎片化的技術實踐 → 抽象為方案模式 → 轉化為可執行的 Agent 技能。我的目標是實現「認知 → 代碼 → 部署」的閉環。

🛠️ 技術矩陣 (Capability Matrix)#

維度 核心能力 (Core Competencies) 關注焦點 (Current Focus)
Depth $\downarrow$ Linux Kernel, eBPF, K8s Internals, HCI, TCP/IP Stack 零拷貝 I/O, 內核級可觀測性, 高性能網路調優
Breadth → LLM-Ops, Agentic Workflow, RAG, Cognitive Architecture 自主 Agent 演進, 記憶體機制, 智能體協作協議
Logic $\Delta$ SRE Rigor, Revenue-first, Learning by Doing 確定性 AI 交付, 認知架構的模組化

⚖️ 執行原則 (Working Principles)#

  • First Principles (第一性原理)：不迷信工具，只相信底層機制。無論是 $\text{io_uring}$ 還是 $\text{Prompt Engineering}$，首先理解其物理/邏輯邊界。
  • High-Rigor Execution (高強度執行)：拒絕「大概」、「差不多」。在生產環境中，唯一的標準是 $\text{Verification (驗證)}$ 與 $\text{Evidence (證據)}$。
  • Closed-Loop Evolution (閉環演進)：實踐 → 失敗 → 分析 → 模式化 → 自動化。

如果你也對系統底層的「黑洞」或 AI Agent 的「湧現」感興趣，歡迎在我的 Hubs 中探索。我們在確定性與概率性的交匯處見。
